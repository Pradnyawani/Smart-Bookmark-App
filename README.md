# Smart Bookmark App

A simple smart bookmark manager built using Next.js, Supabase, and Tailwind CSS.  
Users can log in with Google, add private bookmarks, and see realtime updates.

##  Features
- Google Authentication using Supabase
- Add & delete bookmarks
- Bookmarks are private per user
- Realtime updates (sync across tabs)
- Clean UI with Tailwind CSS

## Tech Stack
- Next.js (App Router)
- Supabase (Auth + Database + Realtime)
- Tailwind CSS
- Vercel (Deployment)
